# Diko

**Diko** est une application d'apprentissage de mots en version bêta, actuellement disponible avec plus de 1000 mots et uniquement en français.

## Fonctionnalités

- **Ajouter des mots** dans différents paquets d'apprentissage (recto : mot et verso : définition avec le lexique et la nature).
- **Choix des définitions** pertinentes pour l'apprentissage.
- **Ajouter des nouveaux paquets**, par thématique par exemple. Présence d'un paquet initial "Mes mots" avec tous les mots en favori.
- **Aprentissage espacé**, basé sur l'algorithme de [SuperMemo2](https://www.supermemo.com/en/blog/application-of-a-computer-to-improve-the-results-obtained-in-working-with-the-supermemo-method) et [Anki](https://faqs.ankiweb.net/what-spaced-repetition-algorithm.html)
